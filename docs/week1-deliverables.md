# Week 1 Deliverables

This week focuses on building the **basic program structure**.

Your program should run and display a working menu.

## Goals

- Create the main game loop
- Implement a console menu
- Use Scanner for user input
- Create the Player object
- Select a game theme

## Required Features

Your program should allow the user to:

1. Create a player
2. Choose a game theme
3. View player information
4. Exit the program

## GitHub Workflow

1. Create a branch: ```week1-structure```
2. Implement your program features.
3. Commit your work regularly.
4. Push the branch to GitHub.

## Canvas Submission

Submit:

- GitHub repository link
- Branch name
- Screenshot of your program running
- Short description of your progress
