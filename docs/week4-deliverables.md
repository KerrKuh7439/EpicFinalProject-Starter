# Week 4 Deliverables
Java EPIC Final Project

Week 4 focuses on **game logic and program state**.

Your program should now move beyond structure and object design and begin behaving like a working system where actions produce meaningful results.

---

# Week 4 Goals

By the end of Week 4 your program should:

• allow users to trigger game actions  
• update player or character state  
• track progress through missions, events, or systems  
• display results of actions clearly in the console  
• handle invalid input safely  

Your program should now feel like a **working application**, not just a collection of classes.

---

Create a new branch for Week 4.
```week4-game-logic```

Commit regularly and push progress to GitHub.

---

# Program Requirements

Add **at least one meaningful game system** that changes program state.

Examples of state changes include:

- health updates
- experience points (XP)
- score changes
- inventory updates
- resource tracking
- mission completion

Your program should:

- perform an action
- update values
- show results to the user
