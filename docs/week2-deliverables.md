# Week 2 Deliverables
Java EPIC Final Project

Week 2 focuses on building the **core object-oriented structure** of your project.

Your program should now begin managing real data using classes and collections.

---

# Week 2 Goals

By the end of Week 2 your program should:

• include additional classes beyond the starter files  
• create objects using constructors  
• store objects in an `ArrayList`  
• allow users to create objects through the menu  
• allow users to display stored objects  

This week transforms your program from a simple menu system into a **data-driven application**.

# Program Requirements

Your program should now manage **multiple objects** using classes.

Required features:

- Create **at least one new class**
- Use **private fields** with getters and setters
- Use **constructors** to create objects
- Store objects in an `ArrayList`
- Add menu options that create objects
- Add menu options that display stored objects

Example collection:

```java
ArrayList<Character> party = new ArrayList<>();
```
