# Final Build
Java EPIC Final Project

This week you will complete and polish your project for final submission.

Your goal is to deliver a **fully working Java application** that demonstrates the programming concepts learned throughout the course.

---

# Final Project Goals

Your completed program should demonstrate:

• multiple interacting classes  
• encapsulation using private fields  
• inheritance and polymorphism  
• collections such as `ArrayList`  
• working game logic and state changes  
• reliable user input handling  

Your program should run **from start to finish without crashing**.

---

# GitHub Workflow

Create the final development branch.
```final-build```

Example: ```git checkout -b final-build```


Complete your final work on this branch.

Once your project is complete:

1. Test your program
2. Fix any remaining bugs
3. Push your final commits
4. Merge the branch into **main**

Example:
```
git checkout main
git merge week5-final-build
git push origin main
```


Your **main branch should contain the final version of your project.**

---

# Final Feature Checklist

Your final project should include:

- working main menu
- player creation
- multiple character types
- subclass behavior (inheritance)
- collections storing objects
- actions that change game state
- clear console output
- input validation

Your system should feel like a **complete small application**.
